# 基于ID3决策树的鸢尾花分类实验

## 实验概述

本实验基于经典的鸢尾花（Iris）数据集，手动实现了一个 **ID3 决策树**分类器，采用**五折交叉验证**对模型性能进行评估，最终平均分类准确率达到 **99.3%**。

---

## 数据集

- **数据来源**：Iris 数据集（`iris.data.csv`）
- **样本总量**：150 条
- **特征维度**：4 个连续型特征
  - `sepal_length`（花萼长度）
  - `sepal_width`（花萼宽度）
  - `petal_length`（花瓣长度）
  - `petal_width`（花瓣宽度）
- **类别标签**：3 类
  - `Iris-setosa`
  - `Iris-versicolor`
  - `Iris-virginica`

---

## 方法

### 1. 数据划分

采用**五折交叉验证**（5-Fold Cross Validation）：

- 按照样本索引对 5 取余的结果，将数据集均匀划分为 5 份。
- 每轮以其中 1 份作为测试集，其余 4 份作为训练集，共进行 5 轮训练与测试。
- 最终以 5 次准确率的均值作为模型的评价指标。

### 2. 决策树构建（ID3 算法）

决策树以**最小化条件熵**作为特征与划分点的选择准则：

**信息熵定义：**

$$H(D) = -\sum_{k} p_k \log_2 p_k$$

**条件熵定义：**

$$H(D|A, t) = \frac{|D^-|}{|D|} H(D^-) + \frac{|D^+|}{|D|} H(D^+)$$

其中 $D^-$ 为特征 $A$ 小于划分点 $t$ 的子集，$D^+$ 为大于等于划分点 $t$ 的子集。

**划分点搜索策略：**

- 对每个特征，从其最小值出发，以步长 `0.1` 遍历至最大值，枚举所有可能的二元划分点。
- 选取使条件熵最小的特征与划分点作为当前节点的分裂依据。
